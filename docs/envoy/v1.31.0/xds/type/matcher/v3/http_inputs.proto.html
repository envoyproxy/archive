[.xds.type.matcher.v3.HttpAttributesCelMatchInput proto]

Specifies that matching should be performed on the set of HTTP attributes.

The attributes will be exposed via Common Expression Language runtime to associated CEL matcher.

Refer to Unified Matcher API documentation for usage details.
